## 功能特性

- 🗺️ **地图集成**：自动解密百度/腾讯坐标并在百度地图上显示位置（需自行申请百度API_KEY）
- 📍 **坐标拦截**：拦截并修改玩家提交的坐标，并自动计算随机偏移量
- 🎯 **自定义答案**：支持选择提交当前点击坐标或真实坐标
- 📏 **距离提示**：显示点击坐标与真实坐标的距离和方向

## 安装方法

1. 安装 Tampermonkey 浏览器扩展（参考教程https://zhuanlan.zhihu.com/p/128453110 ）
2. 打开 Tampermonkey 管理面板
3. 创建新脚本，将本仓库中的 `script.js` 的内容复制进去
4. 保存并启用脚本
5. Enjoy it.

## 使用方法

1. 访问网站
2. 脚本会自动运行，显示浮动窗口
3. 点击右上角的"⚙️ 设置"按钮可以配置功能
4. 当提交答案时，如果启用了自定义回答，会弹出选择窗口

## 配置说明

### 启用百度地图悬浮窗（可选）

在脚本开头的配置区修改 `BAIDU_MAP_AK` 为你的百度地图 API Key。
您需要在这里注册账号并实名认证：https://lbsyun.baidu.com/

### 功能说明

- **启用方向提示**：显示点击坐标与真实坐标的距离和方向信息
- **启用自定义回答**：拦截 guess 请求并显示弹窗，允许选择提交的坐标

## 注意事项

- 需要有效的百度地图 API Key 才能显示地图
- 脚本会在页面加载时自动运行
- 设置会自动保存到浏览器本地存储
- 若没有弹出悬浮窗请多刷新几次
- 目前可在全国赛模式正常使用，部分港澳台/其他地区可能无法解析
